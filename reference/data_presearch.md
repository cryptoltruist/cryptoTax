# Sample data set of a fictive Presearch transaction history file

A fictive Presearch data set to demonstrate
[`format_presearch()`](https://cryptoltruist.github.io/cryptoTax/reference/format_presearch.md).

## Usage

``` r
data_presearch
```

## Format

A data frame with 12 rows and 3 variables:

- date:

  the date

- amount:

  quantity

- description:

  description

## Source

<https://presearch.com/>
