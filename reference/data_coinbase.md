# Sample data set of a fictive Coinbase transaction history file

A fictive Coinbase data set to demonstrate
[`format_coinbase()`](https://cryptoltruist.github.io/cryptoTax/reference/format_coinbase.md).

## Usage

``` r
data_coinbase
```

## Format

A data frame with 5 rows and 10 variables:

- Timestamp:

  the date

- Transaction.Type:

  transaction type

- Asset:

  coin

- Quantity.Transacted:

  quantity

- Spot.Price.Currency:

  currency of spot price

- Spot.Price.at.Transaction:

  spot price

- Subtotal:

  subtotal

- Total..inclusive.of.fees.and.or.spread.:

  Grand total

- Fees.and.or.Spread:

  fees

- Notes:

  comments
