# Sample data set of a fictive BlockFi transaction history file

A fictive BlockFi data set to demonstrate
[`format_blockfi()`](https://cryptoltruist.github.io/cryptoTax/reference/format_blockfi.md).

## Usage

``` r
data_blockfi
```

## Format

A data frame with 15 rows and 4 variables:

- Cryptocurrency:

  currency

- Amount:

  quantity

- Transaction.Type:

  transaction type

- Confirmed.At:

  the date

## Source

<https://blockfi.com/>
