# Sample data set of a fictive Binance transaction history file

A fictive Binance data set to demonstrate `data_binance()`.

## Usage

``` r
data_binance
```

## Format

A data frame with 39 rows and 7 variables:

- User_ID:

  user ID

- UTC_Time:

  the date

- Account:

  account type

- Operation:

  transaction type

- Coin:

  currency

- Change:

  quantity

- Remark:

  comment
