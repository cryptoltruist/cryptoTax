# Sample data set of a fictive CoinSmart transaction history file

A fictive CoinSmart data set to demonstrate
[`format_coinsmart()`](https://cryptoltruist.github.io/cryptoTax/reference/format_coinsmart.md).

## Usage

``` r
data_coinsmart
```

## Format

A data frame with 8 rows and 7 variables:

- Credit:

  amount added

- Debit:

  amount removed

- TransactionType:

  transaction type

- ReferenceType:

  description

- Product:

  the currency

- Balance:

  the current balance

- TimeStamp:

  the date
