# Sample data set of a fictive Celsius transaction history file

A fictive Celsius data set to demonstrate
[`format_celsius()`](https://cryptoltruist.github.io/cryptoTax/reference/format_celsius.md).

## Usage

``` r
data_celsius
```

## Format

A data frame with 12 rows and 9 variables:

- Internal.id:

  internal id

- Date.and.time:

  the date

- Transaction.type:

  transaction type

- Coin.type:

  coin type

- Coin.amount:

  quantity

- USD.Value:

  value in USD

- Original.Reward.Coin:

  original reward coin

- Reward.Amount.In.Original.Coin:

  reward amount in original coin

- Confirmed:

  status of transaction
