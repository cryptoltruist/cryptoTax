# Sample data set of a fictive Adalite transaction history file

A fictive Adalite data set to demonstrate
[`format_adalite()`](https://cryptoltruist.github.io/cryptoTax/reference/format_adalite.md).

## Usage

``` r
data_adalite
```

## Format

A data frame with 10 rows and 11 variables:

- Date:

  the date

- Transaction.ID:

  transaction ID

- Type:

  transaction type

- Received.from..disclaimer..may.not.be.accurate...first.sender.address.only.:

  address the coins were received from

- Received.amount:

  received amount

- Received.currency:

  received currency

- Sent.amount:

  sent amount

- Sent.currency:

  sent currency

- Fee.amount:

  fee amount

- Fee.currency:

  fee currency

- X:

  not used
