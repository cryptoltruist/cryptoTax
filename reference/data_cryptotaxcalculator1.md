# Sample data sets provided by cryptotaxcalculator.io

Data sets from cryptotaxcalculator.io to demonstrate adjusted cost base
as well as capital gains/losses. Used as demo for our own
[`ACB()`](https://cryptoltruist.github.io/cryptoTax/reference/ACB.md)
function.

## Usage

``` r
data_cryptotaxcalculator1

data_cryptotaxcalculator2
```

## Format

A data frame with 4 rows and 4 variables:

- date:

  the date

- trade:

  type of transaction

- currency:

  the coin

- price:

  the spot rate, in dollars

- quantity:

  how much of the stock/coin

An object of class `data.frame` with 4 rows and 5 columns.
