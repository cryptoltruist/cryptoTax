# Sample data sets provided by koinly.io

A data set from koinly.io to demonstrate adjusted cost base as well as
capital gains/losses. Used as demo for our own
[`ACB()`](https://cryptoltruist.github.io/cryptoTax/reference/ACB.md)
function.

## Usage

``` r
data_koinly
```

## Format

A data frame with 4 rows and 4 variables:

- date:

  the date

- transaction:

  buy or sell

- currency:

  the coin

- quantity:

  how much of the stock/coin

- spot.rate:

  the price of the coin
